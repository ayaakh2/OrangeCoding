DOCTYPE

The DOCTYPE declaration is not case-sensitive.

Used to tell the browsers which version of HTML the page is written in.

if we dont use it , the browser will enter quirks mode.

HTML

HTML stands for Hyper Text Markup Language.

HTML is the root element in the page , any other elements are descendants of this element.

head

The head element is used to put the metadata about the HTML page .

Used to define the title of the page, link to stylesheets, JS files .

The data in the head element is not displayed in the browser window .

body

The body element is used to put the content of the HTML page .

All the content that is displayed in the browser window is placed inside the body element , containing all the visible content of the page .

HTML Elements

HTML elements are the building blocks of HTML pages.

HTML elements are represented by tags , Open Tag and Close Tag and the content between them.

Some of html elements are not containing a closing tag , such as <img> and <br> .

Some elements are self-closing and do not require a separate closing tag .

HTML Attributes

HTML attributes provide additional information about HTML elements.

Attributes are always specified in the opening tag.

Attributes usually come in name/value pairs like: name="value".

Some of the common attributes are class ,id ,src , href , alt , and many more .

HTML Paragraphs

HTML paragraphs are defined with the <p> tag.

From the name "paragraph", it is clear that this element is used to define text paragraphs.

HTML Break Line

HTML break line is defined with the <br> tag.

From the name "break line", it is clear that this element is used to break the line of text and move the next content to a new line.

Semantic Elements VS Non-Semantic Elements

Semantic elements clearly describe their meaning to both the browser and the developer.

Semantic elements provide meaning to the content , making it more accessible and understandable.

Non-semantic elements do not clearly define their meaning, just a style .

Block Elements VS Inline Elements

Block elements are used to create larger sections of content , they start on a new line and take up the full width available.

Inline elements are used to format smaller pieces of content , they do not start on a new line and only take up as much width as necessary.

Examples of Block Elements :

  • <div> - Used to group elements together
  • <p> - Used to define paragraphs

Examples of Inline Elements :

  • <span> - Used to group inline elements together
  • <a> - Used to define hyperlinks

HTML Meta Tags

HTML meta tags are used to provide metadata about the HTML document, Dont Displaying to the user .

Meta tags are placed inside the <head> element of the HTML document.

<title>

: Used to define the title of the HTML document, The browser tab title

<meta>

: Used to define metadata about the HTML document,Like character encoding, and viewport settings.

<link>

: Used to link page to external resources,Like CSS files, and favicons.

HTML Comments

HTML comments are used to add notes or explanations to the HTML code, Not Displaying to the user .

HTML comments are placed inside the <!-- --> tags of the HTML document.

HTML comments are not displayed in the browser window, but they can be viewed in the page source code.

HTML comments are useful for documenting the code, and for temporarily disabling code during development.

hr and pre Tags

<hr> is used to define a horizontal line , Used to separate content into sections.

<pre> is used to define preformatted text , Used to display text exactly as it is written in the HTML code.

Text inside a <pre> element is displayed in a fixed-width font, and whitespace is preserved.
